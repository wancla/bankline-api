# Bankline API

### Description
Projeto DIO com parceria do Santander.

### Scop
Realizar a criação de uma API Rest, que irá simular transações bancarias como por exemplo, debito, crédito e extrato bancario.

### Stack
 1. Java 1.8
 2. Jdk 11
 3. Spring Boot 2.6.7
 4. PostgreeSQL 14
